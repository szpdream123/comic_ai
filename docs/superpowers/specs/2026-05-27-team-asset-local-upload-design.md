# 团队资产本地上传设计

## 目标

在团队资产库的角色、场景、道具、音色四个模块中增加本地上传入口。用户上传符合格式的本地文件后，应立即看到成功反馈，并在当前分类下预览上传结果。腾讯云存储后续再接入，本阶段不做真实云端持久化。

## 用户场景

使用者是桌面端创作者和内容生产团队成员，正在一个专业创作工作台中整理团队资产。上传能力应该可见、直接、可信，但不能让未开通团队资产库的锁定态显得自相矛盾。

## 范围

本设计只覆盖团队资产库页面：

- 角色：上传本地图片。
- 场景：上传本地图片。
- 道具：上传本地图片。
- 音色：上传本地音频。

不包含腾讯云存储、真实团队同步、商业权益变更、音色后端表结构调整。

## 推荐方案

采用“本地预览 + 锁定态保留”的混合模式：

- 用户进入团队资产库时始终显示上传入口，包括未开通专业版权益的锁定态。
- 用户上传符合格式的文件后，在锁定面板上方展示“本地上传，待同步”的资产区域。
- 专业版权益锁定面板继续保留在下方，说明团队复用和云端同步仍需要开通团队资产库。

这样用户能马上看到上传结果，同时不会误以为团队资产库已经完整可用。

## 页面布局

上传入口放在分类 tab 下方、资产内容区或锁定面板上方，做成一条紧凑工具栏。

工具栏包含：

- 当前分类名称。
- 一句短状态说明：
  - 角色、场景、道具：本地上传图片用于预览，云端同步稍后接入。
  - 音色：本地上传音频用于预览，云端同步稍后接入。
- 一个主上传按钮：
  - 角色、场景、道具显示“上传图片”。
  - 音色显示“上传音频”。

视觉上沿用现有深色画布、hairline 边框、8px 面板圆角和 pill 按钮体系。上传工具栏应像工作台的一部分，不做营销式横幅。

## 未开通状态

团队资产库未开通时：

1. 先展示上传工具栏。
2. 如果当前分类已有本地上传，展示“本地上传，待同步”区域。
3. 再展示现有专业版权益锁定面板。

锁定文案继续表达：

“团队资产库为专业版会员权益，开通后使用该功能。”

本地上传卡片使用较低强调的状态标签，例如“本地预览”或“待同步”，避免看起来像正式团队资产库记录。

## 上传校验

不符合格式的文件在进入本地状态前拦截。

图片允许格式：

- PNG
- JPG / JPEG
- WEBP

音频允许格式：

- MP3
- WAV
- M4A
- AAC

校验同时参考 MIME type 和文件扩展名，因为浏览器本地文件的 MIME 可能为空或不稳定。

失败反馈使用页面 toast 或轻提示，不使用弹窗：

- 图片类 tab：仅支持 PNG、JPG、WEBP 图片。
- 音色 tab：仅支持 MP3、WAV、M4A、AAC 音频。

不符合格式的文件不能进入上传列表。

## 成功状态

上传成功后：

- 将资产加入当前会话内的本地分类列表。
- 展示成功提示：
  - 图片已上传，可本地预览。
  - 音频已上传，可本地预览。
- 保持当前分类 tab。
- 当前页面立即展示上传卡片。

如果一次选择多个文件，保留所有符合格式的文件，并用简短提示说明有多少文件被拒绝。

## 资产卡片

角色、场景、道具的图片卡片：

- 使用本地 object URL 或 data URL 展示缩略图。
- 展示去掉扩展名后的文件名。
- 展示格式和文件大小。
- 状态标签显示“本地预览”或“待同步”。

音色卡片：

- 展示去掉扩展名后的文件名。
- 展示格式、文件大小，能拿到时展示音频时长。
- 使用原生 audio 控件或紧凑播放条。
- 状态标签显示“本地预览”或“待同步”。

音色卡片不要套用图片缩略图样式，应与视觉资产卡片明显区分，但仍属于同一套资产卡片体系。

## 存储边界

当前实现只保存到前端运行时状态。刷新后不要求保留。

后续腾讯云接入时，用云端返回数据替换本地预览数据：

- 云端 object key。
- CDN 或签名预览 URL。
- MIME type。
- 资产元数据。

前端数据结构应尽量贴近未来云端资产形态，避免二次改造页面。

## 异常和边界

- 没有选择文件：不改变状态。
- 同时选择有效和无效文件：保留有效文件，并提示无效文件被跳过。
- 超过本地批量上限：只加入上限内文件，并提示其余文件被跳过。
- 音频时长读取失败：不阻塞上传，可以隐藏时长或显示占位。
- 切换分类 tab：本次会话内各分类上传列表继续保留。

## 测试

增加前端测试覆盖：

- 团队资产库锁定态仍显示上传工具栏。
- 角色、场景、道具 tab 显示图片上传入口。
- 音色 tab 显示音频上传入口。
- 合法图片文件会进入本地上传状态并显示卡片。
- 合法音频文件会进入本地上传状态并显示卡片。
- 非法图片或音频不会显示成资产卡。
- 本地上传成功后锁定面板仍保留。
